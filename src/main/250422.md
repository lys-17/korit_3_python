## 리스트 인덱싱 / 슬라이싱

리스트 작성은 다음과 같음.
```python
numbers = [1, 2, 3, 4, 5]
# 리스트명 = [요소1, 요소2, 요소3, ...]
```
### 리스트 인덱싱
```python
numbers = [1, 2, 3, 4, 5]
print (numbers[0])
```
인덱싱은 []안에 있는 요소의 값을 말하는 것으로 위처럼 []의 값이 0일 때 0의 자리에 위치해 있는 `1`을 말함

### 리스트 슬라이싱
```python
numbers = [1, 2, 3, 4, 5]
print (numbers[0:3])
print (numbers[:3])
print (numbers[3:])
print (numbers[-3:])
print (numbers[:-2])
```
슬라이싱은 '잘라 낸다'는 것으로
- [0:3] → [1, 2, 3]
- [:3] → [1, 2, 3]
- [3:] → [4, 5]
- [-3:] → [3, 4, 5]
- [:-2] → [1, 2, 3]
위 와 같이 결과값을 나타냄. 여기서 `:`을 기준으로 잘려지는 요솟값이 다름.

### 삼중 리스트에서 슬라이싱
```python
anyList = [[1, "김준일", [3, 4, [7, "김준이"], 6]]]
print (anyList[0][2][2][1])
print (anyList[0][2][1:3])

→ 김준이 / [4, [7, '김준이']]
```
